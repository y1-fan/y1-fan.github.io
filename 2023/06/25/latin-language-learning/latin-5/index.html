拉丁语5：初识形容词与词序
Yi Fan
  2023-06-25 14:45:55 2023-06-25 14:45   2023-07-20 21:21:57  
  • 古文字学
 
  • 拉丁语
  • | 语言基础

一门语言如果没有形容词，那就太平淡无味了。

形容词是用来形容或者描述名词的。在拉丁语中，要求形容词必须与其修饰的名词在性、数、格上保持一致。原则上说，每个形容词都有阳性、阴性和中性三种词尾。

正如名词有多种变格法，形容词的变格法也有分类。这里要介绍的是第一类形容词。这类形容词的词尾，就其阳性单数主格来说分为两种，一种以 -us 结尾，一种以 -er 结尾。同时，它们的阴性形式的词尾都是 -a，中性形式的词尾都是 -um。

在变格的时候需要在阴性单数主格中寻找词根，即去掉 -a 后剩下的部分。然后再按照变格方法在词根之后添加词尾，就得到了不同变格的形容词。变格方法如下（按照阳性、阴性、中性排列）：

  1. 单数
    1. 主格：阳性单词末尾为 -er 或者 -us，阴性单词末尾为 -a，中性单词末尾为 -um。
    2. 属格：词根后添加 -i -ae -i。
    3. 宾格：词根后添加 -um -am -um。
    4. 与格：词根后添加 -o -ae -o。
    5. 夺格：词根后添加 -o -a -o。
    6. 呼格：如果主格末尾为 -us，那么单数呼格变位为词根后添加 -e；如果主格末尾为 -er，那么单数呼格和主格同形。
  2. 复数
    1. 主格：词根后添加 -i -ae -a。
    2. 属格：词根后添加 -orum -arum -orum。
    3. 宾格：词根后添加 -os -as -a。
    4. 与格：词根后添加 -is -is -is。
    5. 夺格：词根后添加 -is -is -is。
    6. 呼格：词根后添加 -i -ae -a。

不难发现，这张变格表基本就是把第一变格法的名词和第二变格法的名词的两张变格表拼接在了一起，这也就是为什么这类形容词也被称为“第一、第二变格法形容词”。

拉丁语的词序实际上相当自由，因为拉丁语高度屈折，很多信息已经包含在词形当中，词序本身并非必要。然而，显而易见的是，一种文化长期以来总是会形成一些语言惯例，拉丁语也不例外。在拉丁语中典型语序是：

  1. 主语及其修饰语
  2. 间接宾语及其修饰语
  3. 直接宾语及其修饰语
  4. 状语
  5. 动词

如果暂时无法熟悉也无需焦虑，在之后的阅读当中会逐渐形成对语序的印象。并且，有大量古罗马文本实际上并不遵循这个惯例，因此，掌握通过词形来解读语句才是重中之重。

词汇

给出一些第一类形容词，按照形态词单数主格的阳性、阴性、中性的顺序给出。

antiquus, antiqua, antiquum 古老的
magnus, magna, magnum 大的；重要的
meus, mea, meum 我的
multus, multa, multum 多的
tuus, tua, tuum （对一个人讲话的时候）你的
avarus, avara, avarum 贪婪的
paucus, pauca, paucium 少的
Romanus, Romana, Romanum 罗马的
bellus, bella, bellum 美的；有魅力的
bonus, bona, bonum 好的；友善的
humanus, humana, humanum 人类的；文明的；精制的
malus, mala, malum 坏的；邪恶的
parvus, parva, parvum 小的
stultus, stulta, stultum 愚蠢的
verus, vera, verum 真实的；正确的

应用

caelicus, caelica, caelicum 天国的

Regna reclude caelica（Veni, veni, Emmanuel）

在上一课中提及的句子中包含了一个第一类形容词。这个词在这里修饰 regna。由上一课知道，regna 是这个中性词的复数主格形式，因此这里也要选择复数主格中性形式，词尾是 -a，故为 caelica。
